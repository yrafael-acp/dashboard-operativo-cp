# Dashboard Operativo CP

Frontend estático listo para GitHub Pages con backend en Google Apps Script.

## Estructura
- `index.html`: shell principal
- `assets/css`: estilos
- `assets/js`: módulos frontend
- `apps-script`: backend API
- `docs`: documentación operativa

## Deploy rápido
1. Desplegar `apps-script/Code.gs` como Web App.
2. Copiar la URL del Web App.
3. Reemplazar `SCRIPT_URL` en `assets/js/config.js`.
4. Publicar este repo en GitHub Pages.

## Desarrollo local
Puedes abrir `index.html` directamente o servirlo con cualquier servidor estático.
