# Flujos operativos

## Dashboard
Carga KPIs, gráfico por zona y tabla general desde `UBICACIONES`.

## Edición
Permite explorar otras hojas, buscar, paginar, guardar, crear y eliminar filas.

## Acciones
- Limpiar A:H
- Limpiar con omisiones
- Consolidado -> Etiquetar
- Etiquetar -> Masivo
- Formato etiqueta
- Refrescar pivots
- Filtrar sin stock
